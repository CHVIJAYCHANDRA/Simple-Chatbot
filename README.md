Simple Chatbot
Overview
This is a Python-based chatbot that responds to user inputs with predefined rules using if-else statements and randomization for varied responses.

Features
Greetings: Responds to greetings like "hello" or "hi".
Farewells: Handles user farewells such as "bye" or "goodbye".
Information: Provides information about the chatbot's identity and capabilities.
Weather: Offers responses related to weather inquiries.
Small Talk: Responds to generic queries like "what are you doing" or "what's up".
Favorites: Shares the chatbot's preferences for color, food (as a bot), and movie.
Dialogues: Provides meaningful dialogues when asked by the user.
Getting Started
To run the chatbot locally:
1.Ensure you have Python installed on your machine.
2.Clone this repository:
git clone https://github.com/your-username/simple-chatbot.git
3.Navigate to the project directory and install any dependencies:
cd simple-chatbot
4.Run the chatbot script:
python chatbot.py
Usage
Type your messages when prompted and press Enter to interact with the chatbot.
The chatbot will respond based on predefined rules and random selections for a more engaging conversation experience.
